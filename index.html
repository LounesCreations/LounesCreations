Comment passer commande ?

Pour passer commande, il vous suffit de nous envoyer un mail a l'adresse mail suivante lounes.creations.off@gmail.com avec les informations suivantes :

1. Nom de l'article : Indiquez le nom de l'article que vous souhaitez personnaliser.

2. Taille : Précisez la taille que vous souhaitez (ex. S, M, L, XL).

3. Couleur : Mentionnez la couleur de l'article.

4. Texte ou Image (facultatif) : Si vous souhaitez ajouter un texte ou une      image      personnalisée, n'oubliez pas de nous donner les détails (texte exact ou fichier image).
